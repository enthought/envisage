import_symbol(symbol_path)[source]¶

Import the symbol defined by the specified symbol path.

‘symbol_path’ is a string containing the path to a symbol through the Python package namespace.

It can be in one of two forms:

  1. ‘foo.bar.baz’

    Which is turned into the equivalent of an import statement that looks like:

    from foo.bar import baz

    With the value of ‘baz’ being returned.

  2. ‘foo.bar:baz’ (i.e. a ‘:’ separating the module from the symbol)

    Which is turned into the equivalent of:

    from foo import bar
eval('baz', bar.__dict__)

    With the result of the ‘eval’ being returned.

The second form is recommended as it allows for nested symbols to be retreived, e.g. the symbol path ‘foo.bar:baz.bling’ becomes:

from foo import bar
eval('baz.bling', bar.__dict__)

The first form is retained for backwards compatability.
